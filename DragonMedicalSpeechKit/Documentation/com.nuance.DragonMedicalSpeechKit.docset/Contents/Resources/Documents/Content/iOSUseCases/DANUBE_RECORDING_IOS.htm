Recording

Prerequisites

You've speech-enabled your app.

Procedure

To start recording, call the startRecording method on the shared session instance: [[NUSASession sharedSession] startRecording:&error];

error â€“ a variable that will receive error information if recording can't be started.

To stop recording, call the stopRecording method on the shared session instance: [[NUSASession sharedSession] stopRecording];

To abort recording, call the abortRecording method on the shared session instance: [[NUSASession sharedSession] abortRecording];

Remarks

  • You must connect a button press event (triggered by a button control or a dedicated hardware button) to the startRecording and stopRecording methods on your NUSASession instance to enable the user to start and stop recording.
  • The abortRecording method discards all unprocessed utterances and pending recognition results.
  • When standby mode is available, stopRecording returns the app to standby mode.
  • When standby mode is unavailable or disabled, stopRecording turns off the microphone completely.
