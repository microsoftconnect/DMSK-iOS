    VAResultCode

    Virtual Assistant v2 error codes.

    • VAResultCodeSuccess
    • VAResultCodeBadRequestError
    • VAResultCodeNetworkError
    • VAResultCodeInternalServerError
    • VAResultCodeApplicationStateError
    • VAResultCodeInternalSDKError
    • VAResultCodeNoSessionError
    • VAResultCodeAutoReconnecting
    } VAResultCode;
    Constants
    VAResultCodeSuccess

    The request has been processed successfully.

    VAResultCodeBadRequestError

    Failed to configure the DMVA. [ConfigurationError+ParameterError].

    VAResultCodeNetworkError

    General network error.

    VAResultCodeInternalServerError

    There was a problem communicating with the DMVA server. The DMVA server is unresponsive or it is sending badly formed data. [ServerError].

    VAResultCodeApplicationStateError

    The application state is not valid for this operation.

    VAResultCodeInternalSDKError

    An unknown error has occurred. [Unknown error+VANotAllowedWithWebViewIntegration error].

    VAResultCodeNoSessionError

    No session error.

    VAResultCodeAutoReconnecting

    Attempts auto reconnecting to VA when the server becomes unreachable.
