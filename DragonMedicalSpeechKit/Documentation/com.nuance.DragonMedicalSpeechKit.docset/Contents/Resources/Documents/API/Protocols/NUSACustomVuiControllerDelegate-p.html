    NUSACustomVuiControllerDelegate Protocol Reference

    <NSObject> : NSObject
    NSObject (NSObject)
    DragonMedicalSpeechKit
    Available in version 1.3 and later
    NUSACustomVuiControllerDelegate.h

    Overview

    NUSACustomVuiController delegate messages.

    Delegate messages sent by the NUSACustomVuiController class to its delegate receiver. All messages provided by the protocol are optional and can be implemented by the receiver. Messages will be sent on the thread that registered for the delegate.

    Instance methods

    speechBarDidShow:

    Sent when the speech bar is shown for a control.
    - (void) speechBarDidShow: (CGRect)frame 
    Parameters
    frame

    Designates the total screen area covered by the speech bar and possibly other input views currently displayed (i.e. the soft keyboard and accessory views, if applicable).

    Discussion

    Sent to the receiver whenever the speech bar is displayed for a control. Provides information about the speech bar's position on the screen.

    Availability
    Available in version 1.3 and later.
