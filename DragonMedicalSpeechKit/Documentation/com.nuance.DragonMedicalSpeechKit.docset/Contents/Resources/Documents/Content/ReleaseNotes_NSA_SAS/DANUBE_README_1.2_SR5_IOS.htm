Release Notes for Nuance SpeechAnywhere 1.2 Service Release 5

This is a service release for the Nuance SpeechAnywhere (iOS edition).

All information in this file is "last minute" and might overrule information in the documentation.

User interface elements

  • The Nuance flame has been removed from all speech-enabled controls.
  • The control with the speech focus is no longer surrounded by a border.
