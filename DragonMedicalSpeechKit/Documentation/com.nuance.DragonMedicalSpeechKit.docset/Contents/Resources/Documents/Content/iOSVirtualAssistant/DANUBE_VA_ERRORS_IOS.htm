Error codes

Dragon Medical Virtual Assistant uses the following error codes as return values for its methods and in NUSAVirtualAssistantControllerDelegate dialogResult, stateChanged and uploadConceptResult methods:

  • NUSAVirtualAssistantErrorCodeNoError - The request has been processed successfully.
  • NUSAVirtualAssistantErrorCodeConfigurationError - Failed to configure DMVA.
  • NUSAVirtualAssistantErrorCodeNetworkError - General network error.
  • NUSAVirtualAssistantErrorCodeServerError - There was a problem communicating with the DMVA server; the server is unresponsive or it's sending badly-formed data.
  • NUSAVirtualAssistantErrorCodeApplicationStateError - The app state isn't valid for this operation.
  • NUSAVirtualAssistantErrorCodeParameterError - The supplied parameters aren't valid.
  • NUSAVirtualAssistantErrorCodeUnknownError - An unknown error has occurred.
