Dragon Medical SpeechKit architecture

 

System components

  • Dragon Medical SpeechKit

    Nuance SDKs to speech-enable your desktop, iOS, Android or browser app.

  • Dragon Medical Server

    Speech recognition servers.

  • Dragon Medical SpeechKit (Browser edition) web server

    Hosts the Browser SDK required to speech-enable browser apps.

  • Nuance Management Center (NMC)

    A web browser interface for administration of Dragon Medical SpeechKit organizations.

    Administrators use NMC to manage user accounts, sites and groups, define auto-texts, manage the speech-recognition vocabulary, monitor system usage and perform other administration tasks. For more information, see the Dragon Medical SpeechKit Administration Guide available from the Nuance Healthcare Development Platform.

  • Nuance Management Server (NMS)

    Provides services for the Dragon Medical system; for example, user authentication. For Dragon Medical SpeechKit, NMS communication is handled by Dragon Medical Server; the client app doesn't communicate directly with NMS.

Network requirements

Dragon Medical SpeechKit connection to Dragon Medical Server: approximately 10 kB/sec upload and download.

On-premises systems

If you are deploying the server-side components on-premises at your own site, see the Dragon Medical Server Installation Guide delivered with your on-premises package for more information on setting up your system.
