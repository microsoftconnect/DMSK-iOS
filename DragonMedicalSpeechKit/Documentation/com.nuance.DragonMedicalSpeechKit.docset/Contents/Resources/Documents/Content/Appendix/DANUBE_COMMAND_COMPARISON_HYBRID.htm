Changes to voice commands

Based on user feedback and technology improvements, voice commands have changed.

Remarks

  • The first voice command listed is the main command, the others are alternatives.
  • This topic lists changes to the Catalan, Hungarian and Slovenian voice commands. For more information on changes to the Danish, Dutch, English, Finnish, French, German, Italian, Norwegian, Portuguese, Spanish and Swedish voice commands, see the 'Changes to voice commands' topic of the integrated end user help.
