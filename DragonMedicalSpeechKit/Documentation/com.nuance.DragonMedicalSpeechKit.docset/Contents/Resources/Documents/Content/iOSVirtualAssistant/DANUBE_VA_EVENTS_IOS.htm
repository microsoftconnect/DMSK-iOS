Events and methods

Your app can be notified about various events that happen during a dialog between the user and Dragon Medical Virtual Assistant by implementing the NUSAVirtualAssistantControllerDelegate and then calling [[NUSAVirtualAssistantController sharedController] setDelegate: ...].

NUSAVirtualAssistantControllerDelegate

The NUSAVirtualAssistantControllerDelegate class contains the following methods:

  • dialogResult - called when DMVA receives a string in JSON format, containing action and task arrays converted from recognized speech. result is the string supplied as the delegate parameter.
  • stateChanged - called when DMVA transitions to the Opened or Closed state. The current state is supplied as the delegate parameter.
  • conceptResult - called when dynamic concept values are uploaded to the DMVA server. The status code and a description of the status code are supplied as delegate parameters.
