Technical restrictions

  • To display your app's end user online help in the personalization & help window, make sure it can be loaded in an iframe. For more information, see: Your application's end user help.
  • The maximum length of a text control is approximately 50,000 characters. Speech recognition might not be available if more than this amount of text is present in all speech-enabled edit controls connected to a VuiController.

Note: Text can be inserted via the keyboard in UITextFields.

  • Sometimes web proxy servers can lack the throughput to properly process Dragon Medical SpeechKit to Dragon Medical Server traffic. This can result in apparent loss of audio, lack of response by the Dragon Medical Recognition Service plus other similar symptoms. We recommend connecting clients directly to the Service instead of going through a web proxy server.
  • UITableView isn't supported by Dragon Medical SpeechKit.
