Sending feedback to DMVA

Call the partner feedback API to send programatically-generated feedback to DMVA. You can also prompt your users for direct feedback based on their experience with DMVA; call the user feedback API to send their feedback.

Partner feedback

For each DMVA intent returned to your app, your integration should call the partnerFeedback API.

- (VAResultCode) partnerFeedback: (VAFeedbackType) feedbackType
                      intentName: (nonnull NSString*) intentName
                           nluId: (nonnull NSString*) nluId
                    feedbackInfo: (nullable NSString*) feedbackInfo
           withCompletionHandler:(void (^ _Nullable)(NSError * _Nullable error))completionHandler;

Pass the following information to the API:

feedbackType - Whether or not your app supports the intent, using the VAFeedbackType enum:

VAFeedbackTypeSupported

VAFeedbackTypeUnsupported

VAFeedbackTypeProcessingError

intentName - The intent name.

nluId - The ID of the DMVA interaction.

feedbackInfo - If applicable, any feedback generated as a result of handling the intent.

completionHandler - If required, a completion handler to receive the return value from DMVA.

User feedback

To send feedback to DMVA from your app's users, implement a GUI to prompt for ratings and comments. Submit the feedback using the userFeedback API.

- (VAResultCode) userFeedback: (nonnull NSNumber*) rating
                 feedbackText: (nullable NSString*) feedbackText
                        nluId: (nullable NSString*) nluId
        withCompletionHandler: (void (^ _Nullable)(NSError * _Nullable error))completionHandler;

Pass the following information to the API:

rating - The user's rating on a scale from 1 (bad) to 5 (excellent).

feedbackText - The user's feedback text, up to 300 characters.

nluId - The ID of the DMVA interaction (nluId).

completionHandler - If required, a completion handler to receive the return value from DMVA.
