Release Notes - 360 | SpeechAnywhere Services 1.3 Service Release 14

This is a service release for 360 | SpeechAnywhere Services iOS edition.

This service release is cumulative and contains all changes to previous service releases.

Solved issues

After an iOS device goes to sleep and then wakes again, speech recognition is now still available.

Speech recognition now continues to work after changing from one wireless network to another while the application is in the background.

After synchronize is called on a custom VUIController, the speech focus no longer changes from the current text control to the first text control in the view.

Previous service release information

Service release 9 (new features)

An abortRecording method has been added; it discards all unprocessed utterances and pending recognition results.

armv7s architecture (iPhone 5) is now supported.

A new skinable GUI element has been added: the speech control. It is an alternative to the speech bar and fully documented in the SDK help.

The minimum requirements have been updated to Xcode 4.5.

Your custom VUI controller integration can now be notified when the speech bar is shown or hidden via the NUSACustomVuiControllerDelegate methods:

speechBarDidShow

speechBarDidHide

Information on the size and position of the speech bar is also returned.

Service release 9 (solved issues)

The speech bar is now positioned correctly in landscape orientation.

Sessions can now be closed cleanly even if recognition results are pending.

The stability of the custom control interface has been improved.

What to do when your application enters the background has been documented.

The speech bar is now always displayed after changing views.

In custom controls, the recognition result is now displayed in the order it was dictated in.

Custom control integration of a web view:

The speech bar is now shown correctly with a split keyboard on an iPad device.

The speech bar now only reacts to input when it is visible.

The speech bar is now shown correctly before recording is started.

Service Release 5 (iOS 6 support)

360 | SpeechAnywhere Services 1.3 Service Release 5 has been tested with iOS 6 Golden Master.

Recording works on devices with iOS 6 installed; it does not work on simulators running iOS 6.
