Release Notes for 360 | SpeechAnywhere Services 1.3 (additional features)

This is a release for 360 | SpeechAnywhere Services (iOS edition).

Replacing the speech bar with your speech recognition controls

360 | SpeechAnywhere Services is delivered with our proprietary speech bar designed for the end user to have feedback about the recognition quality immediately and interact easily with all available speech recognition functionality.

You can disable the speech bar and replace it with your own speech recognition control.

To disable the speech bar, do the following:

  1. Have a class adopt the NUSASpeechBarDelegate protocol.
  2. Use the initWithView:speechEnabledControls:speechBarDelegate: method to initialize a NUSACustomVuiController and pass your class object as speechBarDelegate.

Method signature: (id) initWithView: (UIView*) view controls: (NSArray*) speechEnabledControls speechBarDelegate: (id<NUSASpeechBarDelegate>) speechBarDelegate

See NUSASpeechBarDelegate.h for the declaration of the NUSASpeechBarDelegate protocol.

Temporary words

New words can be added for the duration of a session; for example, patient names can be added and will be recognized correctly.

To add temporary words, use this NUSASession interface method: (void) addTemporaryWords:(NSString*) word spokenForm(NSString*)

Note: The spokenForm is optional and only needs to be specified if it is different from the written form.

Remarks

  • For good recognition quality it is strongly recommended to only add full patient names (do not add first and last names separately). For example, patient Murdo C. Dalziel:

Add:

Murdo C. Dalziel

Murdo Dalziel

Do not add:

Murdo

Dalziel
