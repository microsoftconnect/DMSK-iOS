Dictating into fields

Recording can be started/stopped by your app by explicitly sending the startRecording and stopRecording messages to the shared session object.

When the cursor is placed in a text control and the user starts recording, the resulting text is inserted at the cursor position. When text is selected in a text field and the user starts recording, the selection is replaced by the resulting recognized text.

Navigation

To navigate through the app, the user says next field or previous field; the cursor is placed at the end of the next/previous field in the GUI view and the user can start dictating text again.

Advanced field navigation is also supported; see Enabling named field navigation.
