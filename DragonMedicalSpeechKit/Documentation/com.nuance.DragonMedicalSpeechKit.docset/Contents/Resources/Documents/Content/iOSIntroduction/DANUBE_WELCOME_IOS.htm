Welcome to the Dragon Medical SpeechKit iOS edition help

Dragon Medical SpeechKit enables you to add speech recognition functionality to your iOS apps. We've designed it to be easy for you to integrate and for end users to use.

The service provided by Dragon Medical SpeechKit is guaranteed to be secure and usable wherever the user is working.

For more information on the new features in this version, see the release notes provided with the SDK package.

When your iOS app is speech-enabled, your users can use a variety of our features. For information on the speech recognition features available, what the user sees on screen and how to use the features, see: End user documentation. For information on general concepts, see: Glossary.

You can configure Dragon Medical SpeechKit to connect to Dragon Medical Server set up within your organization’s network, see: On-premises deployment.

Organization of this documentation

This documentation contains the following sections:

  • Concepts - overview - explains the concepts of the product.
  • Setting up Dragon Medical SpeechKit - describes the package that we deliver.
  • Getting started - gives an overview of the steps involved in speech-enabling your app.
  • Dragon Medical SpeechKit design - describes how the product should be used.
  • Dragon Medical SpeechKit architecture - contains a system diagram.
  • Use cases - overview - leads you through speech-enabling your app.
  • Dragon Medical Virtual Assistant - (US Nuance-hosted systems only) how to add virtual assistant functionality to your app.
  • On-premises deployment - describes how to configure the server/client connection if your system is deployed on-premises.
  • Technical restrictions - provides information that you should be aware of when integrating our package.
  • Acknowledgements - references the software of other companies that's included in Dragon Medical SpeechKit.
  • Release History - an archive of previous versions' release notes.
  • API Reference - documents the programming interfaces provided by Dragon Medical SpeechKit.
