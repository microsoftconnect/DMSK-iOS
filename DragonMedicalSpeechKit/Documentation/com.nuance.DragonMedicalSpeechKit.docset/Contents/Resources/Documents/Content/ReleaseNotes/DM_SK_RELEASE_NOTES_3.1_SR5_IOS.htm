Release Notes for Dragon Medical SpeechKit 3.1 Service Release 5 (iOS edition)

Solved issues

  • Auto-texts that include / and " can now be edited. [DNB-8514]
  • Performance speed of the auto-text and vocabulary managers has been improved. [DNB-5455]
  • The SDK version is now available as a comment in header files. [DNB-7712]
  • Dragon Medical-specific swipe gestures, to control recording (swipe-to-speak) and remove text (swipe-to-delete), have been removed. Standard UX guidelines for mobile devices designate the swipe gesture for sliding menus; Dragon Medical SpeechKit no longer interferes with this functionality in your application. [DNB-7914]

  • If a session is not closed explicitly, the iOS edition no longer causes unnecessary network traffic when no recording is running. [DNB-7836]

Important information

  • The WebKit framework also needs added to your application.
  • Dragon Medical SpeechKit 3.1 (iOS edition) is compatible with server versions 2.1, 3.0 and 3.1.
  • Dragon Medical Server 3.1 is compatible with client versions 3.0 and 3.1.
  • For on-premises deployments of Dragon Medical Server with iOS 9 or higher, the connection to Dragon Medical Server will only work if the App Transport Security requirements are met. Unencrypted HTTP connections to servers will no longer work.

What's new in version 3.1

Speech recognition

  • Speech recognition now uses neural network (deep learning) technology. (Only some languages supported.)

Dragon Medical Analytics

  • The Server Management page is now displayed as a carousel in kiosk mode.
  • It is now possible to start Dragon Medical Analytics directly from the Nuance Management Center; this only applies to Nuance-hosted systems.
  • Dragon Medical Server configuration changes and PHI data access are now audited and accessible via the Nuance Management Server Auditor.
  • It is now possible to search for session logs based on the document ID provided by the partner.

User Interface

  • The personalization & help window has been redesigned to improve usability and to prepare it for additional features.
  • Auto-texts with lists are now supported as part of Dragon Medical PowerPacks. The auto-text contains variable text linked to a list of content options.

On-premises systems

  • Dragon Medical Server can now connect to the Microsoft SQL Server using SQL Server Authentication.
  • Dragon Medical Server now supports Microsoft Windows Server 2016. Be aware that Nuance Management Server does not support Windows Server 2016.
  • Italian is now available as a user interface language.
  • Italian is now available as a speech recognition language for non-SpeechMagic-based systems.

Known issues

  • A 16-byte memory leak occurs in the SDK each time a session is closed. [DNB-5441]

  • iPhones on iOS 8: The Manage Vocabulary and Manage Auto-text pages are not displayed properly if rotated to landscape mode while visible; this is due to a known issue in iOS 8. Solution: Upgrade to iOS 9 or 10. [DNB-8483]

Infrastructure

Infrastructure no longer supported

  • Dragon Medical SpeechKit 3.1 applications do not work with version 1.6 servers.
  • In Dragon Medical Server 3.1 all version 1.5 client-based applications stop working.
  • Dragon Medical SpeechKit 3.1 stops support for iOS 7.
  • Dragon Medical SpeechKit 3.1 stops support for Xcode 6 as a development environment.

Infrastructure changes planned for the next release

  • Dragon Medical SpeechKit 3.2 client applications will not work with version 2.0 servers.
  • In Dragon Medical Server 3.2 all version 1.6 and 2.0 client-based applications will stop working.

*End-of-life: third party environments

Support for third party environments is only valid as long as they are supported by the corresponding vendor and might be subject to other restrictions. Please contact Nuance Technical Support for details. For more information, see the documentation delivered with the third party product and supporting Nuance documentation.

When support by the vendor has stopped, Nuance will continue support if an issue is specific to the Nuance solution, within the limitations of the vendor’s end-of-life and Nuance's policy. This means that issues that are a combination of the vendor’s component and the Nuance solution cannot be supported.

Dragon Medical SpeechKit product life cycle

Nuance provides support throughout the life cycle of one version and point release. Please note that Nuance will not perform problem analysis after the scheduled start of restricted maintenance. Dragon Medical Server versions released over three months after the dates stated here will not work with these client editions. For up-to-date information on these schedules, go to Nuance iSupport.
