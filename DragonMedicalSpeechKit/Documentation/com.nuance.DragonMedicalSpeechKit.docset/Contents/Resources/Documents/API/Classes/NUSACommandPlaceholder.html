    NUSACommandPlaceholder Class Reference

    NSObject : NSObject
    NSObject (NSObject)
    DragonMedicalSpeechKit
    Available in version 1.3 and later
    NUSACommandPlaceholder.h

    Overview

    Representation of a placeholder in an application command of an application-defined command set.

    Instance methods

    initWithID:label:

    Initializes a placeholder object.
    - (id) initWithID: (NSString *)id  label: (NSString *)label 
    Parameters
    id

    The unique identifier of the placeholder. This will be returned in the vuiControllerDidRecognizeCommand() delegate message for identification of the placeholder.

    label

    The user-readable label for the placeholder.

    Discussion

    Use this method to create a command placeholder.

    Availability
    Available in version 1.3 and later.

    setValues:withSpokenForms:

    Adds a new set of values to the placeholder.
    - (void) setValues: (NSArray *)values  withSpokenForms: (NSArray *)spokenForms 
    Parameters
    values

    The values that the system will return when it recognizes the associated spoken form. Must have exactly as many members as spokenForms.

    spokenForms

    The spoken forms that the system attempts to recognize. Must have exactly as many members as values. Multiple spoken forms can be defined for a single value by separting them with '|'.

    Discussion

    Availability
    Available in version 1.3 and later.

    clearValues

    Clears all values and associated spoken forms.
    - (void) clearValues
    Discussion

    Availability
    Available in version 1.3 and later.
