Customizing the volume meter

The volume meter provides visual feedback when a user is recording and when the sound quality is poor. You can customize the following:

  • Position on the screen
  • Color
  • Color when the sound quality is poor

Prerequisites

You've speech enabled a GUI view (see: Speech-enabling your iOS app).

Position

To set the volume meter visibility and its position on the screen, set the volumeMeterLocation property of your NUSAVuiController or NUSACustomVuiController object. For example:

[myVuiController setVolumeMeterLocation: NUSAVolumeMeterLocationSide];

Available values

  • NUSAVolumeMeterLocationNone

    Hides the volume meter. This is the default value.

  • NUSAVolumeMeterLocationTop

  • NUSAVolumeMeterLocationBottom

  • NUSAVolumeMeterLocationSide

See also: API reference.

Color

To set the color of the volume meter when the sound quality is OK, set the volumeMeterColor property of your NUSAVuiController or NUSACustomVuiController object. For example:

[myVuiController setVolumeMeterColor: [UIColor blueColor]];

To set the color of the volume meter when the sound quality is poor, set the volumeMeterBadQualityColor property of your NUSAVuiController or NUSACustomVuiController object. For example:

[myVuiController setVolumeMeterBadQualityColor: [UIColor redColor]];

Remarks

  • Choose colors that are easily differentiated.
  • If volumeMeterBadQualityColor isn't set, the volume meter automatically uses the complementary color to volumeMeterColor to indicate poor sound quality. For example, if volumeMeterColor is set to [UIColor cyanColor] then [UIColor redColor] is used by default to indicate poor sound quality.
