NUSAVuiControllerDelegate.h

Overview

NUSAVuiController delegate messages.

Delegate messages sent by the NUSAVuiController class to its delegate receiver. All messages provided by the protocol are optional and can be implemented by the receiver. Messages will be sent on the thread that registered for the delegate.

Tasks

Processing state handling

  • - vuiControllerDidStartProcessing Sent after the end user has started to speak.
  • - vuiControllerDidFinishProcessing Sent after Dragon Medical SpeechKit has finished processing all audio data.
  • - vuiControllerDidStartProcessingForView: Sent after the end user has started a new utterance in a speech-enabled text control￼.
  • - vuiControllerDidFinishProcessingForView: Sent after Nuance SpeechAnyhwere has recognized all utterances started by the end user in a specific text control.￼

Instance methods

vuiControllerDidStartProcessing

Sent after the end user has started to speak.
- (void) vuiControllerDidStartProcessing
Discussion

Sent to the receiver whenever the user starts speaking in a speech-enabled GUI view. Each vuiControllerDidStartProcessing message will be paired with a corresponding vuiControllerDidFinishProcessing message.

Availability
Available in version 1.0 and later.

vuiControllerDidFinishProcessing

Sent after Dragon Medical SpeechKit has finished processing all audio data.
- (void) vuiControllerDidFinishProcessing
Discussion

Sent to the receiver whenever the user stops speaking and Dragon Medical SpeechKit has finished processing all audio data and added all recognized text to the views' text controls. Note that there might be a delay between when the user stops speaking and this message. The same is true if you send the stopRecording (NUSASession) message.

Availability
Available in version 1.0 and later.

vuiControllerDidStartProcessingForView:

Sent after the end user has started a new utterance in a speech-enabled text control￼.
- (void) vuiControllerDidStartProcessingForView: (UIView *)view 
Parameters
view

A pointer to the speech enabled UIView (for NUSAVuiController) or a pointer to the object that implements the NUSATextControl protocol (as passed to NUSACustomVuiController’s initWithView or synchronize methods).

Discussion

Sent to the receiver whenever the user started an utterance in a speech enabled UIView object on your applications user interface. Each vuiControllerDidStartProcessingForView: message will be paired with a corresponding vuiControllerDidFinishProcessingForView: message.

Availability
Available in version 1.0 and later.

vuiControllerDidFinishProcessingForView:

Sent after Nuance SpeechAnyhwere has recognized all utterances started by the end user in a specific text control.￼
- (void) vuiControllerDidFinishProcessingForView: (UIView *)view 
Parameters
view

A pointer to the speech enabled UIView (for NUSAVuiController) or a pointer to the object that implements the NUSATextControl protocol (as passed to NUSACustomVuiController’s initWithView or synchronize methods).

Discussion

Sent to the receiver when Dragon Medical SpeechKit has finished working on all utterances started for a speech-enabled text control. Note that this message does not indicate that the user stopped speaking and a new utterance can be started at any point while recording is on. At the time this message is sent, the text contents of the UIView are already updated with the recognized text.

Availability
Available in version 1.0 and later.
